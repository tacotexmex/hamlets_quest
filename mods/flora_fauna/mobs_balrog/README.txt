
   MOBS BALROG MODULE
   ==================

   Adds a balrog mob.


   Version: 0.2.0
   Source code's license: LGPLv2.1
   Media (Textures, Models, Sounds) license: CC-BY-SA 3.0

   Dependencies: default module (found in minetest_game), mobs (Mobs Redo)

   Report bugs or request help on the forum topic.


   Installation
   ------------
   
   Unzip the archive, rename the folder to mobs_balrog and place it in
   ../minetest/mods/

   GNU+Linux: If you use a system-wide installation place it in
   ~/.minetest/mods/

   If you only want this to be used in a single world, place the folder in
   ../minetest/worlds/world_name/worldmods/

   For further information or help see:
   http://wiki.minetest.com/wiki/Installing_Mods
