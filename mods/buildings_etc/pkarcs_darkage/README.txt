
   PKARCS DARKAGE MODULE
   =====================

   Adds arcs made out of Darkage's nodes.


   Version: 0.1.0
   Source code's license: GPLv3

   Dependencies: pkarcs, darkage (Addi's version)


   Installation
   ------------
   
   Unzip the archive, rename the folder to pkarcs_darkage and place it in
   ../minetest/mods/

   GNU+Linux: If you use a system-wide installation place it in
   ~/.minetest/mods/

   If you only want this to be used in a single world, place the folder in
   ../minetest/worlds/your_worlds_name/worldmods/

   For further information or help see:
   http://wiki.minetest.com/wiki/Installing_Mods
