
	FALLEN NODES MODULE
	===================

	Adds more nodes to the falling_node group.

	Version: 1.4.0
	Source code's license: GPLv3

	Dependencies: default, stairs, farming (found in Minetest Game)
	Supported: landscape, mg, darkage (Addi's version)

	Report bugs or request help on the forum topic:
	https://forum.minetest.net/viewtopic.php?f=11&t=18239


	Installation
	============

	Unzip the archive, place the folder in
	../minetest/mods/

	If you only want this to be used in a single world, place it in
	../minetest/worlds/WORLD_NAME/worldmods/

	GNU+Linux - If you use a system-wide installation place it in
	~/.minetest/mods/

	For further information or help see:
	https://wiki.minetest.net/Help:Installing_Mods
